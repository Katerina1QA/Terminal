1) Посмотреть где я - pwd
2) Создать папку - mkdir folder name
3) Зайти в папку - cd folder name
4) Создать 3 папки - mkdir folder name folder name folder name
5) Зайти в любую папку - cd folder name/
6) Создать 5 файлов (3 txt, 2 json) - cat > name.txt или cat > name.json
7) Создать 3 папки - mkdir folder name folder name folder name
8) Вывести список содержимого папки  - ls -la folder name
9) + Открыть любой txt файл - cat name.txt
10) + написать туда что-нибудь, любой текст. - cat >> name.txt
11) + сохранить и выйти. - control C
12) Выйти из папки на уровень выше - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv name.txt name.txt folder name
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp name.txt name.json folder name
15) Найти файл по имени - cat name.txt
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. - grep sec * 
17) вывести несколько первых строк из текстового файла - head -2 name.txt
18) вывести несколько последних строк из текстового файла - tail name.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.- less name.txt
20) вывести дату и время - date или  date +"%D %T"
